# エンドポイント定義

## 概要

このドキュメントでは、システムが提供する API のエンドポイントについて説明します。  
エンドポイント定義は、各エンドポイントの URL、HTTP メソッドなどを定義します。  
リクエスト/レスポンスの形式やパラメータ、認証方式などの詳細は、OpenAPI YAML ファイルに記載します。
